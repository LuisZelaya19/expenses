## Como usarlo

- Clonar el repositorio usando __git clone__
- Copiar el archivo __.env.example__ a __.env__ y editalo poniendo tus credenciales para la base de datos
- Ejecuta el comando __composer install__
- Ejucuta el comando __php artisan key:generate__
- Ejecuta el comando __php artisan migrate --seed__
- Ahora ya puedes ejecutarlo.
- Puedes iniciar sesion con las siguientes credenciales __admin@admin.com__ - __password__
